# Ujjwal Tiwari | Portfolio

A premium, high-performance portfolio website designed for an Aspiring Data Scientist & Electronics Engineer.

![Portfolio Preview](/public/favicon.svg)

## 🚀 Overview

This project is a modern, responsive portfolio built with **React**, **TypeScript**, and **Tailwind CSS**. It features a custom "Deep Navy & Electric Cyan" theme inspired by data science aesthetics, sophisticated glassmorphism effects, and seamless animations.

## ✨ Features

- **Premium Aesthetic:** Custom-designed "Data Scientist" theme with deep blues and neon accents.
- **Responsive Design:** Fully fluid layout that works perfectly on mobile, tablet, and desktop.
- **Glassmorphism:** Modern UI elements with frosted glass effects.
- **Interactive Components:**
  - **Dynamic Hero Section:** With floating data points and smooth entry animations.
  - **Experience Timeline:** Visual journey of professional internships.
  - **Education & Skills:** Showcasing academic background and technical stack.
- **Performance Optimized:** Built with Vite for lightning-fast loading.

## 🛠️ Tech Stack

- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + Custom CSS Variables
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Package Manager:** npm

## 🏃‍♂️ Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Knight6azer/UTPPS.git
    cd UTPPS
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
